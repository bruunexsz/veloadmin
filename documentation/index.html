Installation

Step 1 — Install PHP Dependencies

composer install

Step 2 — Environment Setup

cp .env.example .env
php artisan key:generate

Step 3 — Install NPM Dependencies & Build Assets

npm install
npm run build

Step 4 — Start Development Server

php artisan serve

Open http://127.0.0.1:8000 in your browser. You should see the Sales Dashboard.

Step 5 — Development with Hot Reload (Optional)

npm run dev
Note: No database is required. VeloAdmin is a frontend template — all data is static.

Theming & Colors

All colors are defined as CSS custom properties in resources/css/veloadmin.css. To change the primary color, simply update --va-primary:

:root {
  --va-primary: #6366f1;      /* Indigo — change this */
  --va-primary-rgb: 99,102,241;
  --va-primary-hover: #4f46e5;
  --va-success: #10b981;
  --va-danger: #ef4444;
  --va-warning: #f59e0b;
  --va-info: #06b6d4;
}
Always update both --va-primary and --va-primary-rgb together to maintain correct opacity effects.

Dark Mode

Dark mode is toggled via the moon/sun icon in the topbar. The theme is stored in localStorage under the key va-theme.

To default to dark mode, change in resources/views/layouts/app.blade.php:

<html lang="en" data-theme="dark" data-bs-theme="dark">

Dark mode overrides are defined in veloadmin.css under [data-theme="dark"].
